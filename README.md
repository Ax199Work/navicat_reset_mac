# 使用说明

## [navicat premium](https://www.navicat.com.cn/download/navicat-premium)
从`navicat-premium`官网下载最新版。安装并运行，选择试用14天。
## 试用脚本
待试用到期后，运行重置脚本。`./reset_navicat.sh`
